## 📞 IF YOU GET STUCK

### Most Common Issues:

**Sheet ID not working**
- Make sure you copied the exact ID from the URL
- No quotes, no spaces, no extra characters

**Apps Script won't deploy**
- Click "Authorize access"
- Grant all permissions
- Try deploying again

**Website not live on GitHub**
- Check repo name is EXACTLY: `selflovesalon.github.io`
- Check `index.html` is in root (not subfolder)
- Wait 3-5 minutes for GitHub Pages to build

**Bookings not appearing in Sheets**
- Check Google Apps Script execution log
- Look for error messages
- Verify Sheet ID is correct

**Each guide has a TROUBLESHOOTING section with more solutions**
