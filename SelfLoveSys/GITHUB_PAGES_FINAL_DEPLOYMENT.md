## 🎯 DEPLOYMENT STEPS (Do These Now)

### Step 1: Create GitHub Account (if you don't have one)
1. Go to **GitHub.com** → https://github.com
2. Click **"Sign up"**
3. Enter email, password, username
4. Verify email
5. Login

### Step 2: Create Repository
1. After login, click **"New"** (top left)
2. **Repository name:** `selflovesalon.github.io` (EXACT - must be this)
3. **Description:** "Self-Love Beauty Salon booking website"
4. Select **"Public"**
5. Check **"Add a README file"**
6. Click **"Create repository"**

### Step 3: Upload Your Files
**Two options:**

#### **Option A: Easy (Web Upload)**
1. In your repo, click **"Add file"** → **"Upload files"**
2. Drag & drop or select:
   - `index.html` ← Most important
   - `testimonials.json`
   - `assets/` folder (entire folder with all images)
3. Scroll down → Click **"Commit changes"**
4. Wait 2 seconds, page will refresh

#### **Option B: Advanced (Git Command)**
```bash

cd C:\Users\lihlo\OneDrive\Desktop\SelfLoveSys

git init
git remote add origin https://github.com/YOUR_USERNAME/selflovesalon.github.io.git
git add .
git commit -m "Initial commit: Self-Love Beauty Salon website"
git branch -M main
git push -u origin main
```

### Step 4: Enable GitHub Pages
1. In your repo, go to **Settings** (tab at top)
2. Scroll down to **"GitHub Pages"** section
3. **Source:** Select **"Deploy from a branch"**
4. **Branch:** Select **"main"** and **"/root"**
5. Click **"Save"**
6. Wait 2-3 minutes

- Refresh page (Ctrl+F5)
- Check repo name is EXACTLY `selflovesalon.github.io`

### Changes not showing
- Wait 2-3 minutes
- Hard refresh (Ctrl+Shift+Delete then refresh)
- Check you clicked "Commit changes"

### Images not loading
- Make sure you uploaded the entire `assets/` folder
- Check file paths in index.html are correct

### Social media buttons don't work
- Make sure links start with `https://`
- No typos in URLs
- Click works (opens in new tab)
